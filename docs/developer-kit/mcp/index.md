# MCP (bnbchain-mcp)

[Model Context Protocol](https://modelcontextprotocol.io/) server for BNB Chain: blocks, transactions, wallets, tokens, NFTs, Greenfield, and ERC-8004 agent tools. Run with `npx @bnb-chain/mcp@latest` in Cursor, Claude Desktop, or any MCP-capable IDE.

**Supported:** BSC, opBNB, Greenfield, and other EVM networks in the toolkit.
